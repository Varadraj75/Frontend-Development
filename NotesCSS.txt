Flex Box : flexible box layout 

It is a layout method for arranging items in a row or column. 

A flexbox always consists of two components : 

1. Flex container : the parent container <div></div>
2. Flex Items : The items inside the container div 

CSS properties used for the flex-container : 

1. flex-direction 
2. flex-wrap 
3. flex-flow 
4. Justify content 
5. align-items
6. align-content 

Flex-direction : specifies the display direction of flex-items in the flex-container 

flex-direction : value can be  (row, column, row-reverse, column-reverse)

Flex-wrap property : specifies whether the flex-items should wrap or not. 

flex-wrap : nowrap, wrap, wrap-reverse

Flex-flow : shorthand property for setting both the flex direction and flex-wrap properties. 

flex-flow : row wrap 

Justify Content : It is used to align the flex-items when do not use all the available space on the main axis horizontal

center 
flex-start 
flex-end 
space-around 
space-between 
space-evenly


Align-Items : It is used to align the flex-items when do not use all available space vertically. 

Align-Items : 

center, flex-start, flex-end, stretch, baseline, normal

ALIGN-CONTENT : is used to align the flex lines 

center, stretch, flex-start, flex-end, space-around, space-between, space-evenly
